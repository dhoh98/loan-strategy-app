# 💰 AI Loan Repayment Strategy Analyzer

## 📌 프로젝트 개요
대출 상환 전략(원리금균등 vs 원금균등)을 비교하고 
총 이자 비용과 월별 상환 스케줄을 시각화하는 웹 애플리케이션입니다.

AI 시대의 개발자 역량을 보여주기 위해
문제 정의 → 전략 설계 → 구현 → 배포까지 전 과정을 수행했습니다.

---

## 🎯 해결하려는 문제

대출을 받을 때 대부분 사용자는:
- 어떤 상환 방식이 유리한지 모름
- 총 이자 차이를 직관적으로 이해하지 못함
- 단순 계산기 이상의 비교 기능이 없음

이 프로젝트는 이를 해결하기 위해
전략별 상환 구조를 시각화하고 총 비용을 비교할 수 있도록 설계되었습니다.
