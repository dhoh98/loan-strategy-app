# 💰 AI Loan Strategy PRO  
### AI 기반 대출 상환 전략 의사결정 시스템

---

## 📖 프로젝트 개요

AI Loan Strategy PRO는  
대출 상환 방식(원리금균등 vs 원금균등)을 단순 비교하는 계산기를 넘어,  

✔ 총 이자 비용  
✔ 초기 상환 부담  
✔ 현금 흐름 안정성  
✔ 전략 점수화 기반 다중 기준 평가  

를 통해 **사용자 맞춤형 상환 전략을 추천하는 AI 의사결정 지원 시스템**입니다.

본 프로젝트는 "AI 시대 개발자 역량 평가 과제"를 위해 제작되었습니다.

---

## 🎯 문제 정의

### 🔎 해결하려는 문제

대출을 받을 때 대부분의 사용자는  
"원리금균등과 원금균등 중 무엇이 유리한가?"라는 질문에 대해  
정확한 판단 기준 없이 선택합니다.

기존 금융 앱들은:

- 단순 월 상환액 계산
- 총 이자 합계 비교

만 제공하며,  
**의사결정을 도와주는 전략적 분석은 부족합니다.**

---

### 🎯 목표

> 단순 계산기를 넘어  
> 사용자 상황을 고려한 전략 판단 도구 제공

---

## 👤 대상 사용자

- 사회초년생
- 주택담보대출 예정자
- 기존 대출 리파이낸싱 고려자
- 금융 의사결정을 데이터 기반으로 하고 싶은 사용자

---

## ⚙ 핵심 기능

### 1️⃣ 상환 방식 비교
- 원리금균등 상환 계산
- 원금균등 상환 계산
- 총 이자 비교

### 2️⃣ 시각화 분석
- 잔액 추이 비교 그래프
- 월 상환액 비교
- 전략 점수 레이더 차트

### 3️⃣ AI 전략 추천
- 총 비용
- 초기 부담
- 안정성 점수화
- 다중 기준 기반 추천

---

## 🧠 전략 판단 로직

추천 알고리즘은 다음 요소를 기반으로 점수를 산출합니다:

- 총 이자 비용
- 첫 달 상환 부담
- 상환 구조 특성

가중치를 적용하여 최종 점수를 계산하고  
더 높은 점수를 받은 전략을 추천합니다.
