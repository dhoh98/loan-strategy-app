# 💰 AI Loan Strategy PRO

대출 상환 전략(원리금균등 vs 원금균등)을 비교 분석하여  
사용자의 의사결정을 돕는 Streamlit 기반 웹 애플리케이션입니다.

---

## 📌 프로젝트 개요

대출을 받을 때 많은 사용자가  
**원리금균등**과 **원금균등** 중 어떤 방식이 유리한지 명확히 이해하지 못합니다.

본 프로젝트는 다음을 직관적으로 보여주는 것을 목표로 합니다:

- 총 이자 비교
- 잔액 감소 추이 비교
- 월 상환액 흐름 비교
- 상세 상환 스케줄 제공

복잡한 금융 계산을 시각화하여  
사용자가 합리적인 선택을 할 수 있도록 돕습니다.

---

## 🚀 주요 기능

### ✅ 핵심 기능
- 원리금균등 / 원금균등 상환 스케줄 자동 계산
- 총 이자 비용 비교
- 잔액 추이 비교 차트
- 월 상환액 비교 차트
- 상세 상환 스케줄 표 제공 (반올림 + 천 단위 콤마 적용)

### 🎯 UX 개선 요소
- 대출 금액 "만원 단위" 입력 지원
- 숫자 가독성 개선 (소수점 제거 + 천 단위 콤마)
- 직관적인 2열 비교 레이아웃 구성

## 🖥 실행 방법 (로컬 환경)

### 1️⃣ 저장소 클론

```bash
git clone https://github.com/yourname/ai-loan-strategy-pro.git
cd ai-loan-strategy-pro
```

### 2️⃣ 패키지 설치

```bash
pip install -r requirements.txt
```

### 3️⃣ 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 실행됩니다.
