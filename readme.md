# 💰 AI Loan Repayment Strategy Analyzer (Advanced)

## 📌 프로젝트 개요

대출 상환 전략(원리금균등 vs 원금균등)을 단순 계산이 아닌
**의사결정 지원 관점에서 비교·분석**하는 웹 애플리케이션입니다.

본 프로젝트는 단순 계산기 구현이 아니라,

- 문제 정의
- 사용자 관점 설계
- 전략 비교 로직 구현
- 대환대출 시뮬레이션
- 의사결정 가이드 제공

까지 포함한 금융 의사결정 지원 도구를 목표로 개발되었습니다.

---

## 🎯 문제 정의

대출을 받을 때 대부분의 사용자는:

- 어떤 상환 방식이 더 유리한지 모름
- 총 이자 차이를 직관적으로 이해하지 못함
- 대환 시 실제 절감액을 계산하지 못함
- 현금흐름 안정성과 총 비용 중 무엇을 우선할지 판단하기 어려움

기존 계산기는 단순 월 납입액만 보여줄 뿐,
전략적 비교 기능은 제공하지 않습니다.

### ✅ 해결 목표

1. 두 상환 전략을 동시에 비교
2. 총 이자 차이 자동 계산
3. 대환대출 절감 효과 시뮬레이션
4. 사용자 상황에 따른 전략 추천

---

## 🚀 주요 기능

### 1️⃣ 상환 전략 비교
- 원리금균등상환
- 원금균등상환
- 총 이자 자동 비교
- 유리한 전략 자동 안내

### 2️⃣ 시각화
- 월별 잔액 변화 그래프
- 전략 간 감소 속도 비교

### 3️⃣ 대환대출 분석
- 새로운 금리 입력 시
- 총 절감액 계산
- 대환 유리 여부 안내

### 4️⃣ AI 기반 전략 가이드
- 상환 기간에 따른 전략 추천
- 현금흐름 vs 총 이자 관점 설명

---

## 🧠 설계 의도

이 프로젝트는 단순한 금융 계산기가 아니라,
**사용자의 선택을 돕는 도구**로 설계되었습니다.

- 계산 → 비교 → 해석 → 추천
의 흐름을 구현하여
사용자가 숫자가 아닌 전략을 이해하도록 구성했습니다.
